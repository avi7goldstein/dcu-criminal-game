# DCU Criminal Game

## Description
The DCU Criminal Game is an engaging and immersive experience where players take on the roles of various characters within a virtual criminal environment, navigating through missions, challenges, and storylines.

## Features
- **Multiple Characters:** Choose from a variety of characters, each with unique abilities and backstories.
- **Interactive Missions:** Undertake missions that affect the game world and narrative.
- **Dynamic Gameplay:** Experience a mix of action, strategy, and decision-making.

## Getting Started
### Prerequisites
- Ensure you have Python 3.8 or higher installed.
- Install necessary libraries using `pip install -r requirements.txt`.

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/avi7goldstein/dcu-criminal-game.git
   ```
2. Navigate to the project directory:
   ```bash
   cd dcu-criminal-game
   ```
3. Follow the setup instructions in the documentation to configure the game.

## How to Play
1. Run the game script using Python:
   ```bash
   python game.py
   ```
2. Follow the on-screen instructions to start your adventure.
